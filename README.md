# Extensive differential gene expression and regulation by sex in human skeletal muscle

This repository contains scripts to generate manuscript figures.

[bulk](https://github.com/schanks/muscle_sex/blob/main/bulk/README.md): Sex differences in bulk gene expression

[composition](https://github.com/schanks/muscle_sex/blob/main/composition/README.md): Sex differences in cell type composition

[miRNA](https://github.com/schanks/muscle_sex/blob/main/miRNA/README.md): Sex differences in miRNA expression

[snATAC](https://github.com/schanks/muscle_sex/blob/main/snATAC/README.md): Sex differences in cell type specific chromatin accessibility (snATAC-seq)

[snRNA](https://github.com/schanks/muscle_sex/blob/main/snRNA/README.md): Sex differences in cell type specific gene expression (snRNA-seq)
